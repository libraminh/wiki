# How to deal with last item when using justify-content: space-between

Use fill-empty class to solve this problem

<div class="wrap-box">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>

  <!-- 4 `fill-empty` boxs = (6 - 2) -->
  <div class="fill-empty"></div>
  <div class="fill-empty"></div>
  <div class="fill-empty"></div>
  <div class="fill-empty"></div>
</div>

.wrap-box {
  display: flex;
  justify-content: space-between;
  flex-wrap: wrap;
}

.box {
  background: pink;
  width: calc(100% / 6 - 10px);
  height: 300px;
  margin-bottom: 15px;
}

.fill-empty {
  width: calc(100% / 6 - 10px);
  height: 0;
}

Demo: https://codesandbox.io/s/css-flexbox-trick-justify-content-space-between-obzoz
